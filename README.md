
# Contract Analysis & Risk Assessment Bot

GenAI-powered contract understanding tool for SMEs and freshers.

## Features
- Clause risk scoring
- Compliance checks (India)
- Plain English explanations
- PDF reports
- Streamlit UI

## Run
pip install -r requirements.txt
streamlit run app.py
